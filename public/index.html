File upload on Cloudinary using Multer

  • Multer :- Multer is a node.js middleware for handling multipart/form-data , which is primarily used for uploading files.
  • Cloudinary :- Cloudinary is a software solution based on Software-as-a-Service (SaaS). It is primarily used for managing all the media assets present in the cloud or Effortlessly optimize, transform, upload and manage your cloud's assets..
  • As developers we need to store our multipart/form-data(Media files like pictures and videos) somewhere, here is the solution for that we can store files on cloudinary by the help of multer(because ExpressJs don't have support multipart data).
  • So by using this website you can upload your file and you will get the URL link for that file after uploading successfully and you can used that URL link in your projects.
  • Frame changes according to your device,open inspect of this web and see the frame.
  • Note :- Don't upload anything that you want to private ( Not everyone can see , but i can see all pictures and videos that upload on this website).
